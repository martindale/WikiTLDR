*Wiki TL;DR* is an extension for Safari / Chrome which reformats WikiLeaks' cablegate documents for optimal reading.

Text is reflowed and capitalized, abbreviations are expanded, maps are added and the entire page is styled. Security clearance requirements are indicated with background colors per paragraph. Message priority is indicated next to each recipient.
